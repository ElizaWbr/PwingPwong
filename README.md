# PwingPwong
Projeto de ping pong criado durante o curso de Desenvolvedor Unity da EBAC.

## Fases
Pwing Pwong possui 3 modos de jogo:
- Um jogador: Player X Bot - Um jogador joga contra um inimigo automático, tentando ter mais precisão do que algo programado para isso, talvez a chave seja a valocidade?
- Dois jogadores: Player X Player - Um jogador enfrenta o outro. O melhor vence.
- Cooperativo - Aqui os dois dão as mãos e tentam alcançar a maior pontuação possível. O meu recorde foi 10 em...

### Dificuldade
Um jogo de pingue-pongue em que a bolinha apenas vai de um lado para o outro é chato. Por isso, ela fica cada vez mais rápida.

O início é fácil..

### Condição de vitória
Para que um jogador ganhe do inimigo automático ou de outro jogador ele precisa:
- Fazer no mínimo 5 pontos;
- Ter pelo menos 3 pontos de vantagem sobre o adversário.

### Controles
- Jogador Um: Setinhas para cima e para baixo;
- Jogador Dois: W e S;

## Rankings
Existem 2 rankings no jogo:
- Top 5 jogadores: a pontuação mais alta de um jogador fica salva com o nome que ele escolheu e aos poucos monta um ranking dos top 5 melhores;
- Top 5 cooperativo: as 5 melhores duplas são exibidas com MUITA honra ao lado de suas super hiper mega altas pontuações, é de dar inveja em quem só quis competir contra o amiguinho;

## Customização
Cada jogador pode escolher um nome para ser melhor representado nos rankings e uma cor para deixar as partidas mais coloridas.
